# HR Analytics Dashboard (Power BI)

## 📊 Overview
This project analyzes employee data to understand attrition trends using Power BI.

## 🧹 Data Cleaning
Data was cleaned using Power Query inside Power BI:
- Removed null values
- Standardized formats
- Created calculated columns

## 📈 Features
- Attrition analysis
- Department-wise insights
- Salary distribution
- Employee demographics
